# Honlor Admin Workspace 🛡️

A premium, high-performance admin dashboard built specifically for **Honlor**, a privacy-first real-time messaging platform. 

This repository contains the custom PHP MVC backend and the Vanilla JS / Tailwind CSS frontend used to manage the Honlor ecosystem. It provides system administrators with the tools needed for real-time user moderation, channel orchestration, ad campaign management, and server health monitoring (integrating Supabase, Firebase FCM, and Tinode).
